# UpUdate Weather Study Case

## Installation

clone from repository
```bash
git clone https://github.com/VeysiVT/upudate-weather-case-study.git
```
install dependencies
```bash
composer install
```

## Configuration

set .env
```bash
cp .env.example .env
```
* You need to add an OPEN_WEATHER_MAP_APPID to .env file


start server
```bash
php artisan optimize:clear
php artisan serve
```
